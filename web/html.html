HyperText Markup Language — základní stavební kámen každé webové stránky

Co je to HTML?

HTML (HyperText Markup Language) je značkovací jazyk pro tvorbu webových stránek. Prohlížeč čte HTML tagy a podle nich zobrazuje obsah — tagy samotné viditelné nejsou.

HTML se kombinuje s:

  • CSS — vzhled a styly
  • JavaScript — interaktivita

Vývoj HTML:

  • 1991 — první verze
  • 2014 — HTML5 (aktuální)

Základní struktura dokumentu

Každá webová stránka má tuto kostru:

<!DOCTYPE html>           <!-- typ dokumentu -->
<html lang="cs">           <!-- jazyk stránky -->
  <head>                   <!-- informace o stránce -->
    <meta charset="UTF-8">
    <title>Název stránky</title>
  </head>
  <body>                   <!-- viditelný obsah -->
    <p>Ahoj světe!</p>
  </body>
</html>

Odkazy a obrázky

Odkaz <a>

<!-- Odkaz na jiný web -->
<a href="https://www.google.com">Google</a>

<!-- Otevřít v novém okně -->
<a href="https://example.com" target="_blank">Odkaz</a>

<!-- E-mail -->
<a href="mailto:info@example.com">Napsat e-mail</a>

<!-- Odkaz na místo na stránce -->
<a href="#sekce1">Přejít na sekci 1</a>
<h2 id="sekce1">Sekce 1</h2>

Obrázek <img>

<!-- Základní obrázek -->
<img src="obrazek.jpg" alt="Popis obrázku">

<!-- Se zadanými rozměry -->
<img src="foto.jpg" alt="Popis" width="300" height="200">

<!-- Obrázek s popisem (sémanticky správně) -->
<figure>
  <img src="foto.jpg" alt="Popis">
  <figcaption>Popisek pod obrázkem</figcaption>
</figure>
Atribut alt je povinný — zobrazí se, když se obrázek nenačte, a pomáhá čtečkám obrazovky pro nevidomé.

Sémantické tagy HTML5

Sémantické tagy říkají prohlížeči a vyhledávačům, co obsah znamená — nejen jak vypadá.

<header>Hlavička stránky / sekce</header>
<nav>Navigační menu</nav>
<main>Hlavní obsah stránky</main>
<article>Samostatný článek</article>
<section>Tematická sekce</section>
<aside>Boční panel / poznámka</aside>
<footer>Patička stránky</footer>
Místo samých <div> používej sémantické tagy — kód bude přehlednější a lépe čitelný pro vyhledávače (SEO).

Tipy pro správný kód

  1. Odsazování — dvě nebo čtyři mezery, důsledně
  2. Správné uzavírání tagů — otevřené tagy vždy uzavři ve správném pořadí
  3. Malá písmena — HTML není case-sensitive, ale malá písmena jsou zvykem
  4. Uvozovky u atributů — hodnoty vždy v uvozovkách: href="..."
  5. Validace — ověř kód na validator.w3.org

Náměty na projekty

  • Stránka o oblíbeném filmu / knize / hře

  • Průvodce místem (město, škola)
  • Jednoduchý blog s více stránkami

Pokročilé formuláře

Formuláře mají mnoho možností, jak zlepšit práci uživatele — seskupení polí, nápovědy i vestavěná validace bez JavaScriptu.

Seskupení polí — <fieldset> a <legend>

<form>
  <fieldset>
    <legend>Osobní údaje</legend>
    <label for="jmeno">Jméno:</label>
    <input type="text" id="jmeno" name="jmeno" required placeholder="Zadej jméno">
    <label for="email">E-mail:</label>
    <input type="email" id="email" name="email">
  </fieldset>
  <fieldset>
    <legend>Třída</legend>
    <input type="radio" id="r7" name="trida" value="7">
    <label for="r7">7. třída</label>
    <input type="radio" id="r8" name="trida" value="8">
    <label for="r8">8. třída</label>
  </fieldset>
  <button type="submit">Odeslat</button>
</form>

Nápovědný seznam — <datalist>

<label for="mesto">Město:</label>
<input type="text" id="mesto" list="mesta">
<datalist id="mesta">
  <option value="Praha">
  <option value="Brno">
  <option value="Ostrava">
</datalist>
<datalist> zobrazí nápovědu při psaní — uživatel si může vybrat, nebo napsat vlastní hodnotu.

Meta tagy v hlavičce

Meta tagy jsou v sekci <head> a poskytují informace o stránce — prohlížeči, vyhledávačům i sociálním sítím.

<head>
  <!-- Kódování znaků — vždy jako první! -->
  <meta charset="UTF-8">

  <!-- Správné zobrazení na mobilech -->
  <meta name="viewport" content="width=device-width, initial-scale=1.0">

  <!-- Popis stránky (zobrazuje se ve výsledcích vyhledávání) -->
  <meta name="description" content="Výukové materiály z informatiky pro ZŠ">

  <!-- Klíčová slova (dnes méně důležitá) -->
  <meta name="keywords" content="html, css, informatika, škola">

  <!-- Autor stránky -->
  <meta name="author" content="David">

  <title>IT Návody</title>
</head>
Meta tag description se zobrazuje přímo ve výsledcích vyhledávání pod názvem stránky — piš ho pečlivě, je to první věc, co uživatel vidí.

Kompletní struktura webové stránky

Ukázka dobře napsané HTML stránky s využitím sémantických tagů — třídní web jako příklad:

<!DOCTYPE html>
<html lang="cs">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta name="description" content="Stránka třídy 7.A">
  <title>Třídní web 7.A</title>
</head>
<body>

  <header>                    <!-- hlavička s logem a navigací -->
    <h1>Třídní web 7.A</h1>
    <nav>
      <a href="index.html">Úvod</a>
      <a href="rozvrh.html">Rozvrh</a>
      <a href="kontakt.html">Kontakt</a>
    </nav>
  </header>

  <main>                      <!-- hlavní obsah stránky -->
    <article>
      <h2>Aktuality</h2>
      <p>Příští týden je výlet na hrad Karlštejn.</p>
    </article>
    <aside>                  <!-- boční panel -->
      <h3>Rychlé info</h3>
      <p>Třídní učitel: pan Novák</p>
    </aside>
  </main>

  <footer>                    <!-- patička -->
    <p>&copy; 2025 Třída 7.A</p>
  </footer>

</body>
</html>

Přístupnost (Accessibility)

Přístupný web mohou používat všichni — i lidé se zrakovým postižením, kteří používají čtečky obrazovky.

Pro pokročilé: ARIA atributy a skip-link

ARIA (Accessible Rich Internet Applications) atributy přidávají sémantiku tam, kde HTML nestačí:

<!-- Popis pro čtečku (neviditelný text) -->
<button aria-label="Zavřít dialog">×</button>

<!-- Skrytí dekorativního prvku před čtečkou -->
<img src="dekorace.png" alt="" aria-hidden="true">

<!-- Skip link — přeskočení navigace na hlavní obsah -->
<a href="#obsah" class="skip-link">Přejít na obsah</a>
<main id="obsah">...</main>

Datové atributy data-*

Datové atributy umožňují uložit vlastní data přímo v HTML elementu — bez speciálních tříd nebo skrytých polí.

<!-- Data uložená přímo v HTML -->
<li data-id="42" data-kategorie="sport">Fotbal</li>

<button data-akce="smazat" data-id="42">Smazat</button>

<!-- CSS může data-* použít jako obsah -->
<span data-tooltip="Klikni pro více info">?</span>
Pro pokročilé: čtení dat z JavaScriptu 
const btn = document.querySelector('button');

// Čtení hodnot
console.log(btn.dataset.akce);   // "smazat"
console.log(btn.dataset.id);     // "42"

// Nastavení hodnoty
btn.dataset.stav = 'aktivni';
Atribut data-moje-data se v JS stane dataset.mojeData — pomlčka se převede na camelCase.

SVG v HTML

SVG (Scalable Vector Graphics) jsou vektorové obrázky přímo v HTML. Na rozdíl od JPG nebo PNG se nerozmazají při jakémkoli zvětšení.

<svg width="200" height="100">

  <!-- Obdélník -->
  <rect x="10" y="10" width="80" height="50"
        fill="blue" stroke="black" />

  <!-- Kruh -->
  <circle cx="150" cy="35" r="25"
           fill="red" />

  <!-- Čára -->
  <line x1="0" y1="75" x2="200" y2="75"
        stroke="black" stroke-width="2" />

  <!-- Text -->
  <text x="10" y="95" font-size="12">Ahoj SVG!</text>

</svg>
PrvekKlíčové atributyPopis
<rect>x, y, width, heightObdélník
<circle>cx, cy, rKruh (střed + poloměr)
<line>x1, y1, x2, y2Úsečka
<text>x, y, font-sizeText
SVG se styluje pomocí CSS nebo přímo atributy: fill = výplň, stroke = obrys, stroke-width = tloušťka obrysu.
