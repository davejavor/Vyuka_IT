GitHub

Tvorba webových stránek pomocí GitHubu

Krok za krokem: od registrace po živý web na internetu

Fáze 1 — Příprava účtu

Krok 1: Vytvoření GitHub účtu

1

Jdi na github.com a klikni Sign up.

2

Zadej e-mail, heslo a uživatelské jméno.

3

Ověř e-mail kódem, který ti přijde do schránky.

Zvol si smysluplné uživatelské jméno — bude součástí adresy tvého webu! Například jan-novak.

Krok 2: Speciální repozitář pro web

1

Po přihlášení klikni na + v pravém horním rohu a vyber New repository.

2

Název repozitáře musí být přesně v tomto formátu:

tvojeuzivatelskejmeno.github.io
Tento přesný formát aktivuje GitHub Pages. Každé písmenko je důležité!
3

Nastav repozitář na Public (veřejný) a zaškrtni Add a README file.

4

Klikni Create repository.

Fáze 2 — Vytvoření souborů webu

Krok 3a: Soubor index.html

1

V repozitáři klikni Add file → Create new file.

2

Jako název souboru napiš index.html.

3

Vlož tento základní HTML kód:

<!-- Základní struktura stránky -->
<!DOCTYPE html>
<html lang="cs">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Můj první web</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <h1>Ahoj světe!</h1>
  <p>Můj první web je online.</p>
</body>
</html>
4

Níže na stránce napiš popis změny (tzv. commit zprávu): Přidán základní HTML soubor a klikni Commit new file.

Krok 3b: Soubor style.css

1

Znovu Add file → Create new file, tentokrát název style.css.

2

Vlož základní styly:

/* Styly pro celou stránku */
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

body {
  font-family: Arial, sans-serif;
  max-width: 800px;
  margin: 2rem auto;
  padding: 0 1rem;
  background: #f4f4f4;
}
3

Commit zpráva: Přidán základní CSS styl → Commit new file.

Fáze 3 — Publikování webu

Krok 4: Aktivace GitHub Pages

1

V repozitáři klikni na Settings (Nastavení).

2

V levém menu najdi sekci Pages.

3

U Source vyber Deploy from a branch.

4

Vyber větev main a složku / (root), pak klikni Save.

Krok 5: Zobrazení webu

1

Počkej 1–3 minuty na zpracování.

2

Tvůj web bude dostupný na adrese:

https://tvojeuzivatelskejmeno.github.io
Gratulujeme! Tvůj web je živý na internetu. 🎉

Fáze 4 — Úpravy a rozšiřování

Jak upravovat web

1

Jdi do repozitáře a klikni na soubor, který chceš upravit (např. index.html).

2

Klikni na ikonu tužky (Edit this file).

3

Proveď změny, napiš popis (commit zprávu) a klikni Commit changes.

4

Počkej minutu, obnov stránku — změny jsou viditelné.

Přidávání obrázků

1

Vytvoř složku images/: Add file → Create new file, napiš název images/placeholder.txt.

2

Nahraj obrázky přes Add file → Upload files.

3

V HTML odkazuj na obrázek takto:

<img src="images/muj-obrazek.jpg" alt="Popis obrázku">

Více stránek

Vytvoř nové soubory (o-mne.html, galerie.html, …) a propoj je pomocí odkazů:

<a href="o-mne.html">O mně</a>

Řešení častých problémů

Problém Řešení
Web se nezobrazuje Zkontroluj název repozitáře — musí být uzivatel.github.io. Zkontroluj aktivaci GitHub Pages v Settings. Počkej 5–10 minut.
Změny se neprojevují Počkej 1–2 minuty po commitu. Tvrdé obnovení: Ctrl+F5 (Windows) nebo Cmd+Shift+R (Mac).
Chyby v kódu Otevři nástroje pro vývojáře v prohlížeči klávesou F12 a podívej se na záložku Console.

Kam dál?

  • Vlastní doména — propojení domény (např. mojestranka.cz) s GitHub Pages
  • JavaScript — přidání interaktivity (kalkulačky, hry, animace)
  • Responsive design — web fungující správně i na mobilu
  • Kolaborace — společné projekty pomocí Pull Requests
  • SEO — optimalizace pro vyhledávače
