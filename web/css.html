Cascading Style Sheets — jak webové stránky vypadají

Co je to CSS?

CSS (Cascading Style Sheets) určuje vzhled HTML stránek — barvy, velikosti, rozmístění a animace. HTML = obsah a struktura, CSS = vizuální podoba.

Výhody CSS:

  • Oddělení obsahu od vzhledu
  • Jeden soubor pro celý web
  • Rychlejší načítání stránek
  • Responzivní design

Vývoj:

  • 1996 — CSS1
  • 1998 — CSS2
  • dnes — CSS3 (modulární)

Jak připojit CSS k HTML

1. Externí soubor (doporučeno)

<!-- V sekci <head> -->
<link rel="stylesheet" href="styl.css">

2. Interní CSS (v hlavičce HTML)

<style>
  body { background: #f0f0f0; }
</style>

3. Inline CSS (přímo v tagu)

<h1 style="color: blue; font-size: 24px;">Nadpis</h1>
Vždy preferuj externí soubor — je nejpřehlednější a platí pro celý web najednou.

Selektory

Selektor říká, na které HTML prvky se pravidlo vztahuje.

/* Typ prvku */
p { color: red; }

/* ID — pouze jeden prvek na stránce */
#hlavni-nadpis { font-size: 32px; }

/* Třída — více prvků */
.zvyrazneny { font-weight: bold; }

/* Všechny prvky */
* { margin: 0; padding: 0; }

Kombinátory

/* Potomek — všechny <p> uvnitř <div> */
div p { text-indent: 20px; }

/* Přímý potomek */
div > p { border: 1px solid black; }

/* Pseudo-třída — při najetí myší */
a:hover { text-decoration: none; }

*Všechny prvky* { … }

Barvy a pozadí

p {

  color: rgba(255, 0, 0, 0.5);       /* RGB + průhlednost */
}

div {

Text a typografie

body {
  font-family: Arial, sans-serif;    /* rodina písma */
  font-size: 16px;                   /* velikost */
  font-weight: bold;                  /* tučnost: normal, bold, 100–900 */
  font-style: italic;                /* kurzíva */
  line-height: 1.5;                  /* výška řádku */
}

  text-align: center;               /* left | right | center | justify */
  text-decoration: underline;        /* podtržení */
  text-transform: uppercase;        /* VELKÁ PÍSMENA */
  letter-spacing: 2px;              /* mezery mezi písmeny */
}

Box model

Každý HTML prvek je obdélníkový box se čtyřmi vrstvami:

+---------------------------+
|         margin            |  ← vnější okraj
|  +---------------------+  |
|  |       border        |  |  ← ohraničení
|  |  +--------------+   |  |
|  |  |   padding    |   |  |  ← vnitřní odsazení
|  |  |  +--------+  |   |  |
|  |  |  | obsah  |  |   |  |  ← samotný obsah
|  |  |  +--------+  |   |  |
|  |  +--------------+   |  |
|  +---------------------+  |
+---------------------------+
div {
  width: 300px;                       /* šířka obsahu */
  padding: 20px;                      /* vnitřní odsazení */
  border: 2px solid black;            /* ohraničení */
  margin: 30px;                       /* vnější okraj */
  box-sizing: border-box;            /* padding a border zahrnuty do width */
}

Pozicování prvků

.static   { position: static; }    /* výchozí — v toku stránky */
.relative { position: relative;    /* posunutý od původní pozice */
             top: 20px; left: 30px; }
.absolute { position: absolute;    /* vůči nejbližšímu pozicovanému předkovi */
             top: 0; right: 0; }
.fixed    { position: fixed;      /* vůči oknu, zůstane i při scrollování */
             top: 10px; right: 20px; }
.sticky   { position: sticky;     /* kombinace relative + fixed */
             top: 0; }

Flexbox

Moderní způsob rozmístění prvků v jedné ose (řádek nebo sloupec).

/* Kontejner */
.flex-container {
  display: flex;
  flex-direction: row;              /* row | column */
  justify-content: space-between;  /* rozmístění na hlavní ose */
  align-items: center;             /* zarovnání na příčné ose */
  flex-wrap: wrap;                  /* zalomení na další řádek */
  gap: 1rem;                        /* mezery mezi položkami */
}

/* Položka */
.flex-item {
  flex-grow: 1;                     /* jak moc může narůst */
  flex-basis: 200px;               /* výchozí velikost */
}
justify-contentChování
flex-startNa začátek (výchozí)
flex-endNa konec
centerNa střed
space-betweenRovnoměrně, krajní na okrajích
space-aroundRovnoměrně, mezery i na okrajích

CSS Grid

Dvourozměrný systém pro rozvržení stránek — řádky i sloupce zároveň.

/* Kontejner */
.grid {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;  /* 3 sloupce v poměru 1:2:1 */
  grid-template-columns: repeat(3, 1fr);/* 3 stejně široké sloupce */
  gap: 1rem;
}

/* Položka přes více sloupců/řádků */
.wide {
  grid-column: 1 / 3;                 /* od 1. do 3. čáry = 2 sloupce */
  grid-column: span 2;                /* totéž, jiný zápis */
}

Transformace a přechody

Transformace

div {
  transform: translate(50px, 20px);  /* posun */
  transform: rotate(45deg);          /* otočení */
  transform: scale(1.5);             /* zvětšení */
}

Plynné přechody (transitions)

button {
  background: blue;
  transition: background 0.3s ease;   /* co | jak dlouho | průběh */
}
button:hover {
  background: darkblue;
}

Animace

/* Definice klíčových snímků */
@keyframes posun {
  0%   { transform: translateX(0); }
  100% { transform: translateX(200px); }
}

/* Použití animace */
.animovany {
  animation: posun 2s ease-in-out infinite alternate;
}

Media Queries — responzivní design

Různé styly podle velikosti obrazovky (mobil, tablet, počítač).

/* Výchozí styl (pro velké obrazovky) */
body { font-size: 16px; }

/* Pro obrazovky do 600px (mobily) */
@media (max-width: 600px) {
  body { font-size: 14px; }
  .menu { display: none; }
}

/* Tisk */
@media print {
  nav, footer { display: none; }
}

/* Tmavý režim */
@media (prefers-color-scheme: dark) {
  body { background: #222; color: white; }
}

Tipy a triky

Centrování prvků

/* Blokový prvek horizontálně */
.box { margin: 0 auto; width: 80%; }

/* Flexbox — horizontálně i vertikálně */
.centrovany {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 300px;
}

/* Grid — zkrácený zápis */
.grid-center {
  display: grid;
  place-items: center;
}

CSS proměnné

/* Definice v :root (platí pro celou stránku) */
:root {
  --barva-primarni: #1a56a0;
  --mezera: 1rem;
}

/* Použití */
h1 { color: var(--barva-primarni); }
.box { padding: var(--mezera); }
Proměnné ti ušetří čas — barvy a rozměry změníš na jednom místě pro celý web.

Responzivní obrázky bez media queries

img {
  max-width: 100%;
  height: auto;
}

Barvy v HSL a HSLA

HSL je alternativní způsob zápisu barev — Hue (odstín), Saturation (sytost), Lightness (světlost). Snáze se přemýšlí o barvách než u HEX nebo RGB.

p {
  /* hsl(odstín 0-360, sytost %, světlost %) */
  color: hsl(0, 100%, 50%);       /* červená */
  color: hsl(120, 100%, 35%);     /* zelená */
  color: hsl(240, 100%, 50%);     /* modrá */

  /* hsla — jako hsl, navíc průhlednost (0–1) */
  color: hsla(0, 100%, 50%, 0.5); /* poloprůhledná červená */
}

/* Výhoda HSL: lehce tvoříme tónové varianty */
.svetla  { background: hsl(220, 70%, 80%); } /* světle modrá */
.tmava   { background: hsl(220, 70%, 30%); } /* tmavě modrá */

Z-index — vrstvení prvků

Když se prvky překrývají, z-index určuje, který bude nahoře. Funguje jen na pozicovaných prvcích (position: relative/absolute/fixed/sticky).

.pozadi {
  position: absolute;
  z-index: 1;    /* dole */
}

.obsah {
  position: absolute;
  z-index: 10;   /* uprostřed */
}

.modal {
  position: fixed;
  z-index: 100;  /* vždy nahoře — modální okno */
}
Vyšší číslo = blíže k uživateli. Záporné číslo (z-index: -1) prvek posune za normální tok stránky.
Z-index funguje jen u prvků s position nastaveným na jinou hodnotu než static.

Float a Clear

Float byl dlouho hlavní způsob rozmísťování prvků vedle sebe — dnes ho nahrazuje Flexbox a Grid, ale stále se používá pro obtékání textu kolem obrázků.

/* Obrázek s obtékajícím textem */
img.vlevo {
  float: left;     /* text obtéká zprava */
  margin: 0 1rem 1rem 0;
}

img.vpravo {
  float: right;    /* text obtéká zleva */
  margin: 0 0 1rem 1rem;
}

/* Clear — ukončení obtékání */
.konec-float {
  clear: both;   /* prvek začne pod floatovanými prvky */
}

/* Clearfix — moderní řešení přetékajícího rodiče */
.clearfix::after {
  content: "";
  display: table;
  clear: both;
}
Pro rozvržení layoutu stránky dnes vždy preferuj Flexbox nebo Grid. Float používej pouze na obtékání textu kolem obrázku — to je jeho původní účel.

Moderní responzivní design

Moderní CSS funkce umožňují vytvořit responzivní design bez jediné media query — prohlížeč si sám vypočítá správné hodnoty.

Funkce clamp() — plynulá typografie

h1 {
  /* clamp(minimum, ideální, maximum) */
  font-size: clamp(1.5rem, 5vw, 3rem);
  /* min 1.5rem, roste s šířkou okna, max 3rem */
}

Funkce min() a max()

.box {
  /* min() vybere menší hodnotu */
  width: min(500px, 90%);   /* nikdy víc než 500px a zároveň max 90% */

  /* max() vybere větší hodnotu */
  padding: max(1rem, 3vw);  /* aspoň 1rem, nebo 3% šířky */
}

Viewport jednotky

.hrdina {
  width:  100vw;  /* 100 % šířky okna */
  height: 100vh;  /* 100 % výšky okna */
  font-size: 4vw; /* 4 % šířky okna */
}

Automaticky responzivní grid

.karty {
  display: grid;
  /* Automaticky přizpůsobí počet sloupců */
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  gap: 1rem;
}
auto-fit + minmax() je nejelegantnější způsob mřížky karet — sám si vypočítá, kolik sloupců se vejde, aniž bys napsal jedinou media query.
