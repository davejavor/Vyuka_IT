IP adresy & DNS

Jak funguje internet — tahák pro průzkum  ·  Navazuje na: Rádiová síť micro:bit

Klíčové pojmy

PojemCo to je
IP adresaJednoznačná adresa zařízení v síti. Jako číslo domu.
IPv44 čísla 0–255, např. 142.250.185.46
IPv6Novější formát: 2001:db8::1 (víc adres)
DNSPřeloží doménu → IP. Jako telefonní seznam.
PaketMalý kousek dat s adresou odesílatele + příjemce.
RouterZařízení, které pakety přeposílá správným směrem.
TTLTime To Live — kolik hopů ještě paket smí projít.
TracerouteZobrazí cestu paketu hop po hopu přes routery.

Typy DNS záznamů

ZáznamCo dělá
Adoména → IPv4 adresa
AAAAdoména → IPv6 adresa
MXdoména → mailový server
CNAMEalias (www → example.com)
TXTlibovolný text (ověření, SPF…)

Jak DNS přeloží doménu

  1. Zadáš google.com do prohlížeče
  2. DNS resolver (škola / ISP) se zeptá Root serveru
  3. Root odkáže na .com server
  4. .com server odkáže na Google NS
  5. Google NS vrátí IP: 142.250.…
  6. Výsledek se uloží do cache na dobu TTL
