Co je micro:bit?

Micro:bit je malý počítač s LED displejem (5 × 5 diod), dvěma tlačítky (A a B) a spoustou senzorů. Programujeme ho v prohlížeči — žádná instalace není potřeba.

Přední strana

  • Tlačítko A — vlevo
  • Tlačítko B — vpravo
  • LED displej — 5 × 5 červených diod uprostřed

Zadní strana

  • USB konektor — napájení a nahrávání programu
  • Reset tlačítko — restartuje program
  • Konektor na baterie

MakeCode editor

  1. Otevři editor

    V prohlížeči jdi na makecode.microbit.org a klikni na Nový projekt.

  2. Orientace v editoru

    Editor má tři hlavní části:

    • Simulátor (vlevo) — virtuální micro:bit, kde vidíš výsledek
    • Kategorie bloků (uprostřed) — Basic, Input, Music…
    • Pracovní plocha (vpravo) — sem přetahuješ bloky

  3. Dva výchozí bloky

    Každý nový projekt obsahuje:

    • on start — spustí se jednou při zapnutí
    • forever — opakuje se pořád dokola

Postup: První program

  1. Zobraz ikonu při startu

    Z kategorie Basic přetáhni blok show icon do bloku on start.

    Vyber si ikonu (srdíčko, smajlík…). Po spuštění se zobrazí na simulátoru.

    ← ikona Heart

  2. Tlačítko A — zobraz text

    Z kategorie Input přetáhni blok on button A pressed na pracovní plochu.

    Dovnitř vlož blok show string "Ahoj" z kategorie Basic.

    💡 Tip: show string zobrazuje text po písmenech (roluje). show number zobrazuje číslo. Vyzkoušej obojí!

  3. Tlačítko B — zobraz obrázek

    Přetáhni další blok on button B pressed.

    Dovnitř vlož show leds z kategorie Basic — tady si můžeš naklikat vlastní vzor na mřížce 5 × 5.

  4. A + B najednou

    Přetáhni on button A+B pressed (najdeš v rozbalovacím menu bloku — klikni na A a změň na A+B).

    Dovnitř vlož clear screen (Basic) — displej se smaže.

  5. Nahrání do micro:bitu

    Připoj micro:bit USB kabelem k počítači.

    Klikni na Stáhnout (Download) dole v editoru.

    • Chromebook / nový MakeCode: Klikni na … vedle tlačítka Download → Connect device → vyber micro:bit → program se nahraje přímo.
    • Starší postup: Stažený soubor .hex přetáhni do složky MICROBIT (zobrazí se jako USB disk).
    ⚠️ Pozor: Při nahrávání micro:bit krátce bliká — počkej, až přestane. Neodpojuj USB kabel během nahrávání!

Přehled užitečných bloků

Blok Kategorie Co dělá
show icon ♥ Basic Zobrazí předdefinovanou ikonu
show leds Basic Zobrazí vlastní vzor na LED mřížce
show string "…" Basic Zobrazí text (roluje po displeji)
show number 42 Basic Zobrazí číslo
clear screen Basic Vypne všechny LED diody
pause (ms) 1000 Basic Počká zadaný čas (1000 ms = 1 sekunda)
on button A pressed Input Spustí kód po stisku tlačítka A
on button B pressed Input Spustí kód po stisku tlačítka B
on button A+B pressed Input Spustí kód po stisku obou tlačítek najednou

Zadání

Vyber si úroveň, která ti sedí. Základ je povinný — rozšíření a výzva jsou pro ty, kdo chtějí víc.

⭐ Základ

  • Tlačítko A zobrazí tvoje jméno (show string)
  • Tlačítko B zobrazí vlastní obrázek (show leds)
  • Tlačítka A + B smažou displej (clear screen)
  • Nahraj program do micro:bitu a ověř, že funguje

🔷 Rozšíření

  • Přidej animaci — po stisku tlačítka se vystřídá několik obrázků za sebou (použij show leds + pause)
  • Použij blok forever k opakující se animaci (blikající srdce, běžící puntík…)
  • Prozkoumej blok show arrow — vytvoř kompas ukazující různé směry

🟣 Výzva

  • Vytvoř počítadlo: tlačítko A přičte 1, tlačítko B odečte 1, na displeji se zobrazí aktuální číslo (budeš potřebovat proměnnou z kategorie Variables)
  • Přepni editor na JavaScript (přepínač nahoře) a zkus program upravit v kódu
  • Najdi v kategorii Input blok on shake — co by mohl micro:bit dělat, když s ním zatřeseš?

Řešení problémů

Micro:bit se nezobrazuje jako USB disk

Zkus jiný USB kabel — některé kabely jsou jen nabíjecí (nemají datové vodiče). Zkus taky jiný USB port na počítači.

Program se nenahrál / nefunguje

Zkontroluj, že jsi soubor .hex přetáhl do správné složky (MICROBIT, ne do jiné). Po nahrání micro:bit restartuje — počkej, až přestane blikat.

Text na displeji je moc rychlý / pomalý

Mezi bloky show leds nebo show icon vlož blok pause a nastav čas v milisekundách (500 = půl sekundy).

Chci smazat blok

Přetáhni blok zpátky do kategorie (doleva) nebo na něj klikni pravým tlačítkem a zvol Delete Block.
