Digitální stopa a metadata

Co o tobě prozrazují soubory, které sdílíš — pro žáky 7.–9. ročníku

Co jsou metadata?

Metadata jsou data o datech — informace, které popisují obsah souboru, aniž by byly jeho přímou součástí.

Analogie: zásilka

  • Obsah zásilky = vlastní soubor (fotka, text, hudba)
  • Obal zásilky = metadata (kdo poslal, kdy, odkud, velikost…)

Obal vidí všichni, i když obsah zůstane zapečetěný.

Kde metadata vznikají?

  • Fotoaparát / telefon — GPS poloha, čas, model přístroje
  • Textový editor — autor, čas vzniku, počet úprav
  • Audio přehrávač — název skladby, interpret, album
  • PDF a obrázky — software, profily barev, licence

Typy metadat podle formátu

Formát Typ metadat Typické informace
JPEG, TIFF, HEIC EXIF GPS, datum, model foťáku, expozice
MP3, FLAC, OGG ID3 Interpret, album, rok, obal
PDF, obrázky Adobe XMP Autor, software, licence, klíčová slova
DOCX, XLSX, PPTX Metadata dokumentu Autor, revize, komentáře, čas úprav

EXIF metadata ve fotkách

EXIF (Exchangeable Image File Format) jsou metadata uložená přímo uvnitř souboru fotografie. Obsahují přesné informace o tom, kdy, kde a čím byla fotka pořízena.

EXIF pole Co obsahuje Příklad
GPS Latitude/Longitude Přesná poloha pořízení 50.0755° N, 14.4378° E
DateTime Datum a čas pořízení 2024:03:15 14:32:07
Make + Model Výrobce a model zařízení Apple iPhone 15 Pro
Software Aplikace, která fotku zpracovala Instagram 292.0
Orientation Natočení přístroje při focení Rotate 90 CW

Jak zobrazit EXIF metadata

1
Windows — Vlastnosti souboru: Klikni pravým tlačítkem na fotku → Vlastnosti → záložka Podrobnosti. Uvidíš GPS, datum, model přístroje i nastavení expozice.
2
Online — jimpl.com/exif: Přetáhni fotku do prohlížeče. Soubor se nenahrává na server — vše probíhá lokálně v prohlížeči. Funguje na Chromebooku i PC.
3
Telefon — vypnout geolokaci: Otevři Nastavení fotoaparátu → hledej Poloha nebo Geolokační značky → vypni. Nové fotky GPS data nebudou obsahovat.
Pozor — sdílení souboru EXIF nemaže: Pokud fotku pošleš přímým přenosem (AirDrop, email, USB, WhatsApp), příjemce dostane i GPS souřadnice a všechna ostatní metadata. Sociální sítě (Instagram, Facebook) EXIF odstraní pro ostatní uživatele, ale samy si ho uchovávají pro vlastní analýzu.
Tip — hromadné smazání EXIF na Windows: Označ více fotek najednou → pravý klik → Vlastnosti → záložka Podrobnosti → odkaz Odebrat vlastnosti a osobní informace ve spodní části okna.

Jak smazat EXIF

  • Windows Vlastnosti: Vlastnosti → Podrobnosti → Odebrat vlastnosti a osobní informace → zvolit, co smazat → OK
  • Online — verexif.com: Nahraj fotku, stáhni verzi bez EXIF dat. Vhodné i na Chromebooku.

Metadata v dokumentech

Textové dokumenty si pamatují mnohem víc, než jen text. Tato metadata mohou odhalit, kdo dokument vytvořil, jak dlouho na něm pracoval, nebo jaké komentáře byly smazány.

Google Docs — Historie verzí

  • Otevři dokument → Soubor → Historie verzí → Zobrazit historii verzí
  • Vidíš každou uloženou verzi s časem a autorem
  • Lze obnovit starší verzi
  • Historie je viditelná všem, kdo mají přístup k dokumentu

Word / DOCX — pro zvídavé

  • Přejmenuj soubor .docx na .zip
  • Rozbal archiv — uvnitř je složka docProps/
  • Otevři soubor core.xml v textovém editoru
  • Najdeš: autora, čas vzniku, čas poslední úpravy, počet revizí

Co se v dokumentech ukládá:

  • Autor — přezdívka nebo jméno z nastavení aplikace
  • Čas vzniku a čas poslední úpravy
  • Počet revizí — kolikrát byl soubor uložen
  • Smazané komentáře — mohou zůstat skryté v souboru
  • Jména spolupracovníků v historii verzí

Zadání pro žáky

Základ

  • Vezmi vlastní fotku (z telefonu nebo PC) a otevři ji na jimpl.com/exif v prohlížeči. Co tam vidíš? Jsou v ní GPS souřadnice?
  • Zkus smazat EXIF data z fotky pomocí verexif.com. Zkontroluj výsledek — co zmizelo?

Rozšíření

  • Otevři libovolný Google Docs dokument → Soubor → Historie verzí → projdi historii. Kdo a kdy dokument upravoval?
  • Na telefonu najdi nastavení fotoaparátu a vypni ukládání GPS souřadnic do fotek. Vyfot testovací snímek a ověř, že poloha v EXIF chybí.
  • Stáhni libovolný soubor .docx, přejmenuj ho na .zip, rozbal a otevři docProps/core.xml. Co se tam skrývá?

Výzva

  • Porovnej metadata různých typů souborů — JPG vs PNG vs PDF. Které formáty ukládají GPS polohu a které ne? Proč?
  • Nainstaluj knihovnu Pillow v Pythonu a přečti EXIF vlastním skriptem: 
    from PIL import Image, ExifTags

img = Image.open("fotka.jpg")
exif = img._getexif()
if exif:
    for tag_id, val in exif.items():
        tag = ExifTags.TAGS.get(tag_id, tag_id)
        print(f"{tag}: {val}")
  • Na Windows PC zkus příkaz v PowerShellu pomocí nástroje exiftool: 
    exiftool fotka.jpg
    Vypíše všechna metadata včetně GPS. Lze hromadně: exiftool *.jpg
